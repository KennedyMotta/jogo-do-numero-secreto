# 🎯 Jogo do Número Secreto

Um jogo desenvolvido com JavaScript onde o objetivo é descobrir um número secreto gerado aleatoriamente. A cada tentativa, o sistema informa se o número digitado é maior ou menor que o valor correto, ajudando o jogador a encontrar a resposta.

## 🌐 Acesse o projeto

https://kennedymotta.github.io/jogo-do-numero-secreto/

![Jogo do Número Secreto](preview.png)

## 🚀 Funcionalidades

- Geração aleatória de número secreto
- Validação dos palpites do jogador
- Dicas para encontrar o número correto
- Contagem de tentativas
- Reinício da partida

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📚 Conceitos praticados

- Manipulação do DOM

- Funções em JavaScript
- Eventos de interface
- Geração de números aleatórios
